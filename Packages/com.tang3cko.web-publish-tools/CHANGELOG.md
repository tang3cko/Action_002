## [0.1.0] - Unreleased

### Added
- Initial release
- Profile-based workflow with `IPublishProfile` interface and `PublishProfileRegistry`
- `UnityroomProfile` as the first supported profile
- Setting cards: Build Target, Compression Format, Decompression Fallback, Development Build, Data Caching, Exception Support, Managed Stripping Level, Resolution
- Resolution card with dropdown of common landscape / portrait / square presets
- UI Toolkit window at `Window > Web Publish Tools`
- BEM-style USS with design tokens
