# キャラクター設定ドキュメント — Polarity Survivors

## ビジュアルコンセプト

「対等な二極」。白も黒も主従なく、分離していることで世界は成立している。
プレイヤーはその境界を越える唯一の存在であり、その越境自体が脅威として造形される。

---

## ブランドカラー（PolarityColors.cs 準拠）

| 役割 | HEX | 使用場面 |
|------|-----|---------|
| White foreground | `#F0EEE6` | White極性エンティティの本体色 |
| White background | `#1A1A2E` | White極性ステージの背景、Black側エンティティのアウトライン |
| Black foreground | `#1A1A2E` | Black極性エンティティの本体色 |
| Black background | `#F0EEE6` | Black極性ステージの背景、White側エンティティのアウトライン |

**設計原則：White極性エンティティはダークネイビー背景に明るく浮かぶ。Black極性エンティティはオフホワイト背景に暗く沈む。**

---

## 技術的制約（デザインに直結する制約）

- **レンダリング：** DrawMeshInstanced + URP Unlit Shader（敵・弾丸）/ SpriteRenderer（ボス）
- **テクスチャ方式：** PNG透過 (アルファチャンネル必須)。アルファクリップ `Cutoff=0.1` で抜く
- **解像度：** ゲーム画面 960×540 WebGL。スプライトは64×64〜256×256推奨
- **色はシェーダーが乗算適用：** テクスチャはグレースケール（白一色）で作り、シェーダーが極性カラーを乗算する
  - 例：Texture2D上は白 `#FFFFFF`、実際の表示色はPolarityColors.GetForeground()で決まる
  - **→ 画像生成では「白いシルエット、黒い透過背景」を基本とする**

## プレイヤーキャラクター

**コンセプト：** 秩序の破壊者。両方の極性を行き来できる唯一の存在。
**造形基盤：** 太極図（陰陽マーク）。体そのものが陰陽の象徴。
**サイズ：** 画面上 約32×32px（スプライト原寸 128×128推奨）
**アングル：** 真上からのトップダウン

### 固定属性

| 属性 | 内容 |
|------|------|
| 基本シルエット | 太極図の円形。S字の境界線が本体の中央を分割 |
| White極性時 | 本体：`#F0EEE6` のS字曲線域が前面。回転：反時計回り |
| Black極性時 | 本体：`#1A1A2E` のS字曲線域が前面。回転：時計回り |
| 共通ディテール | 中央の小円（対極色）がアイポイントとして機能 |
| アウトライン | 対極色でやや太め（DrawMeshInstanced outline pass） |

## 敵キャラクター（モブ）

**共通規則：**
- 幾何学形状ベース。陰陽モチーフを形状で表現する
- テクスチャはグレースケール白シルエット → シェーダーが色を乗算
- 各タイプは**シルエットだけで判別できる**ことが必須条件
- アウトラインは対極色で描画（White敵 → ネイビーアウトライン、Black敵 → オフホワイトアウトライン）

### Shooter（単発狙い撃ち）

| 属性 | 内容 |
|------|------|
| シルエット | 正三角形。一辺が常にプレイヤー方向を向く矢じり型 |
| 陰陽的意味 | 単一の意志を持つ鋭角。分裂していない一点集中 |
| サイズ | 64×64 |

### NWay（螺旋3発）

| 属性 | 内容 |
|------|------|
| シルエット | 三つ巴（みつどもえ）的な3方向回転形状。三角形の各頂点に曲線的な羽が付いた形 |
| 陰陽的意味 | 3方向への拡散。螺旋する弾丸を予感させる旋回感 |
| サイズ | 64×64 |

### Ring（環状8発）

| 属性 | 内容 |
|------|------|
| シルエット | 中心が空洞の八角形リング。内側に細いスリットが8等分で入る |
| 陰陽的意味 | 全方位への拡散と環（輪廻）。中央の空洞が「境界の空白」を表す |
| サイズ | 64×64 |

### Anchor（高速連射、ボス級）

| 属性 | 内容 |
|------|------|
| シルエット | 陰陽魚（いんようぎょ）の片身シルエット。太極図を構成する2尾の魚形のうち1尾だけを取り出した形。丸く膨らんだ頭部と、S字弧を描いて細くなる尾部からなる縦長の有機的形状。頭部中央に対極色の小円（目）を持つ |
| 陰陽的意味 | 対を失った片身。陰陽が分離したことで生まれた「半分の存在」。螺旋弾を放ち続けるのは、失われた片身を探す衝動の表れ |
| サイズ | 96×96 |

**コンセプト：** 勾玉（まがたま）形状。古代の魂の象徴。
**技術：** DrawMeshInstanced + アルファクリップ

| 属性 | 内容 |
|------|------|

## ボス第1形態（白の守護者 / 黒の守護者）

**コンセプト：** 阿吽構造。2体で1つの完全体。片方だけでは機能しない対の存在。
**技術：** SpriteRenderer（DrawMeshInstancedではない）
**サイズ：** 256×256推奨（ステージの1/4程度の視覚的存在感）

### シルエット設計方針

人型・甲冑型ではなく、**背の高いスライム型の不定形シルエット**。
底部が最も幅広く、上方に向かって緩やかに細くなる縦長の柱状流体。
表面は不規則にうねり、液体が引力に逆らって立ち上がるような動的な輪郭を持つ。
頭部（上端）に対極色の円（眼）が浮かぶ。人型・骨格・甲冑的な要素は一切持たない。

### 白の守護者（WhiteGuardian）

| 属性 | 内容 |
|------|------|
| シルエット | 底部が広く上方に細くなる縦長スライム型。不定形・流動的な輪郭 |
| 体色 | `#F0EEE6` ベースのフラットフィル |
| 頭部（上端） | 内部に対極色（`#1A1A2E`）の大きな円が浮かぶ（眼＝対極の目） |
| アウトライン | ネイビー `#1A1A2E` の太いアウトライン（4〜5px相当） |
| 口の状態 | 開口（阿形：「阿」）。正面下部に開いた開口部 |
| 印象 | 穏やかで静的。生きた白い液体が立ち上がった存在 |

### 黒の守護者（BlackGuardian）

| 属性 | 内容 |
|------|------|
| シルエット | 白の守護者と同一のスライム型シルエット。色のみ異なる |
| 体色 | `#1A1A2E` ベースのフラットフィル |
| 頭部（上端） | 内部に対極色（`#F0EEE6`）の大きな円が浮かぶ |
| アウトライン | オフホワイト `#F0EEE6` の太いアウトライン（4〜5px相当） |
| 口の状態 | 閉口（吽形：「吽」）。正面下部が閉じた圧縮された状態 |
| 印象 | 厳格で動的。生きた黒い液体が引力に逆らって立つ存在 |

### 対の関係性

- 2体は左右対称に配置される（白＝左、黒＝右、または逆）
- シルエット構造はほぼ同一。違いは体色・アウトライン色・頭部内円の色・口の状態のみ
- これにより「同じ存在の二面」であることを視覚的に表現する
- 人型を排除することで、存在そのものが「極性エネルギーの液体的な体現」であることを示す

## ボス第2形態（勾玉合体：Magatama）

**コンセプト：** 2体が融合し、陰陽が渦を巻く最終形態。
**技術：** SpriteRenderer
**サイズ：** 512×512（画面の約半分を占める圧倒的存在感）

| 属性 | 内容 |
|------|------|
| シルエット | 巨大な勾玉。`#F0EEE6` と `#1A1A2E` の渦状パターンが表面を覆う |
| 内部パターン | 太極図のS字ラインが複数層重なり、回転しているように見える動的構造 |
| 頭部の名残 | 2つの小円（守護者の「眼」）が渦の中に残存。支配されつつも消えていない |
| スケール感 | 単体でステージを半分覆う重量感 |

## 背景

**共通規則：** 極性変化に連動して背景全体のトーンが切り替わる。ポストプロセスシェーダーは使用不可。

| ステージ | 背景色 | 演出要素 |
|---------|--------|---------|
| White極性ステージ | `#1A1A2E` ダークネイビー | 波紋状の明るい流れ。陰陽的な流動感 |
| Black極性ステージ | `#F0EEE6` オフホワイト | 渦状の暗い流れ。収束するエネルギー感 |
| 境界ステージ（終盤） | グラデーション：中央から両端へ白黒が溶け合う | 境界のにじみ。白黒の侵食表現 |

## メタテーマとビジュアル対応表

| テーマ | 視覚的表現 |
|--------|-----------|
| 対等な二極 | 白も黒も同じシルエット構造。色のみが異なる |
| 分離＝秩序 | 背景色の明確な分離。境界線は鋭くシャープ |
| 越境＝脅威 | プレイヤーだけが両極性を持つ。太極図＝両方を内包する唯一の形 |
| 封印＝分離の維持 | ボス守護者のアウトライン：対極色で「包み込む」構造 |
| 阿吽構造 | 守護者2体の開口/閉口。シルエット一致、色だけ反転 |
| 浸食と抵抗 | 終盤背景のにじみ表現 |
| 境界の曖昧化 | 境界ステージのグラデーション崩壊表現 |
