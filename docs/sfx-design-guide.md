## 方向性

### ビジュアルとの整合
- 陰陽モチーフから**幾何学的・抽象的**な方向に転換済み
- 和楽器ベース（鈴、太鼓、篳篥）は合わなくなった
- **機械的・金属的なスイッチ音**（「ガコン」系）が方向性として適切

### 音のイメージ
- 「ガコン」— 何かが切り替わった感じのある機械的・金属的な音
- 重すぎず軽すぎず、テンポよく連打しても不快にならない
- 長さ 0.2〜0.4秒
- WebGLで短いSFXにノイズが乗る可能性があるため、極端に短い音は避ける

## 有名ゲームのSFX設計から学んだ原則

### 直接参考になる知見

#### SEKIROのパリィ設計 — 「音でプレイヤーを教育する」
- 正しいプレイ（パリィ）= 大きく高い金属音
- 間違ったプレイ（ブロック）= しょぼい音
- → **極性転換に適用**: 自発的な転換は気持ちいい音、ボスの強制転換は不穏で重い音

#### 桜井政博のヒットストップ三位一体
- 打撃SE + ヒットストップ + 画面エフェクトが揃って初めて手触り
- → SFXだけ良くしても効果は半減。視覚フィードバックとの同期が重要

#### モンハンの「足す」vs「引く」
- MH World: レイヤリング（複数素材を重ねて深み）
- MH Rise: 単発のインパクトを凝縮
- → **弾幕ゲームは同時発生が多いのでRise方式（単発凝縮）が適切**

#### ピッチエンベロープの基本
- 高→低 = 「ドスン」（重い、強制転換向き）
- 低→高 = 「ピュー」（上昇感）

#### BGMとの棲み分け
- BGMが使わない帯域にSFXを置く
- サイドチェインでSFX再生時にBGMを一瞬下げる

### 普遍的な原則
1. **フィードバックとしてのSFX** — プレイヤーに「何が起きたか」「良かったのか悪かったのか」を伝える
2. **リアルさより誇張** — ゲーム的インパクトの最大化（桜井氏、ストリートファイターII）
3. **沈黙の価値** — 常に音が鳴っている状態は音の価値を下げる（Dark Souls）
4. **短く、クリアに、即座に** — ミリ秒単位で判断に使われる。長すぎる音、曖昧な音は機能しない
5. **音量は情報の重要度で決める** — 派手だから大きい、ではない（桜井氏）
6. **制約をスタイルに転化する** — 予算・技術的制約は個性になりうる（Undertale、初代マリオ）

## 音楽理論的な根拠

### 波形と感情
| 波形 | 倍音構成 | 感情的印象 |
|------|---------|-----------|
| サイン波 | 基音のみ | 穏やか、純粋 |
| 三角波 | 奇数倍音（弱い） | 柔らかい |
| 矩形波 | 奇数倍音（強い） | 荒い、エッジ感 |
| 鋸歯状波 | 全倍音 | 攻撃的、明るい |

→ 「ガコン」系には**矩形波**が出発点として適切（エッジ感 + 金属的倍音）

### フィルタースイープと「ちゅいん」
- レゾナンスフィルターのカットオフ周波数を高速スイープさせた音
- 倍音のピーク位置が移動することで独特の「うねり」が生まれる
- 「ちゅいん」系は幾何学ビジュアルとの相性は良いが、今回は「ガコン」方向を選択

### ADSR と打撃感
- **短いAttack + 急速Decay + Sustain 0** = パーカッシブ（打撃感）
- Attack速度がゲームフィールに直結
- 金属音は通常打楽器より若干長いDecayで「リング」を表現

### 心理音響学
- 人間の耳は 2.5〜5kHz に最も敏感（等ラウドネス曲線）
- 高周波 = 緊急性・注意喚起
- 低周波 = 脅威・重さ
- → スイッチ系の「カチッ」は 1500〜3000Hz 帯域が適切

## Logic Pro Alchemy での「ガコン」SE 制作手順

### UI構成（ADVANCED モード）
- 上部: BROWSE / SIMPLE / **ADVANCED**
- ソース: A / B / C / D（各ソースにVA/Additive/Spectral/Formant/Granular/Samplerの合成方式）
- フィルター: LP/BP/HP等、12dB/24dB/60dB、Sharp/Smooth/Edgy/Rich/Clean/Gritty等のバリエーション
- モジュレーション: LFO / AHDSR / MSEG / SEQUENCER / ENV FOLLOWER / MODMAP
- マスター: Vol / Pan / Coarse Tune / Fine / VOICES

### パラメータ設定（根拠付き）

| パラメータ | 設定値 | 根拠 |
|---|---|---|
| **OSC** | Square, VA | 金属音の基本波形。TR-808ハイハット回路と同じ出発点（Baratatronix） |
| **演奏ピッチ** | C4〜C5 | スイッチ系の「カチッ」感はこの帯域。C2以下だと重い金属板になる |
| **Filter Type** | **BP 6dB Sharp** | LP 24dBより金属的共鳴をピンポイントで強調可能（MusicRadar, KVR Forum）。LPは音を「暗く」する方向で金属的明るさが失われやすい |
| **Cutoff** | ノブ50%付近（2000〜3000Hz） | スイッチ系トランジェントの帯域（daviddumaisaudio） |
| **Resonance** | 50〜70% | 自己発振直前で共鳴ピークが最大化されパーカッシブなバーストが生まれる（Signal Flux） |
| **Drive** | 20〜30% | 軽めに金属のエッジを追加。上げすぎると歪みが支配的になる |
| **Attack** | 0ms | 打撃音の基本。即座に立ち上がり（NI, LANDR） |
| **Hold** | 0ms | 不要 |
| **Decay** | 100〜150ms（ノブ15〜25%） | ベル系（300-500ms）より短く。「ガコン」の切れ味を出す |
| **Sustain** | 0% | 打撃音なので持続しない |
| **Release** | 30〜80ms | 余韻を少しだけ残す |
| **Voices** | 1 | 単発インパクト音にはモノフォニックで十分 |

### 金属感の強化オプション
- **2nd OSCの追加**: Source BにもSquare波を設定し、+5〜+7セミトーンまたは+40セントでデチューン → 非整数比の倍音で金属感が増す（LMMSフォーラム）
- **極短ディレイ（1〜5ms）+ フィードバック高め** → コムフィルター効果で金属的共鳴を追加（MusicRadar）
- **Comb PMフィルター**（Source Filter内） → バイポーラフィードバックで複数の金属共鳴ピーク生成（Apple公式）。使う場合はFeedbackを0からゆっくり上げること

### 書き出し設定
- 使う部分だけリージョンで選択 → File → Bounce
- **WAV, 44100Hz, 16bit, モノラル**
- 先頭と末尾の無音をトリミング
- ファイル名: `polarity_switch.wav`

### 確認ポイント
- 0.2〜0.4秒に収まっているか
- 連続で5〜6回素早く再生しても不快にならないか
- BGM（120BPM）と一緒に鳴らしても邪魔にならないか
- WebGLビルドで正常に再生されるか（短すぎるとノイズが乗る）
