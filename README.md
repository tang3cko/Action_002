# Action_002 - 極性切り替えリズムSTG

unity1week「うら」投稿作品。ゲーム開発160時間目、制作時間約20時間。

## 概要

白/黒の極性を切り替えながら、同色弾を吸収・異色敵を撃破するバレットヘル。120BPMのビートに同期してプレイヤーはダウンビートで攻撃、敵はオフビートで攻撃する。

## 制作時間の内訳

- 総制作時間: 約20時間
- BGM/SFX制作（Logic Pro初触り含む）: 約10時間
- ゲームデザイン + 実装: 約10時間

## 振り返り

### 機能したもの

- **極性切り替えの初見インパクト**: 1ボタンで6つの同時フィードバック（ヒットストップ、プレイヤー拡大パルス、カメラズーム、衝撃波リング、背景色反転、SFX）が発生し、初見の掴みは機能した。プレイヤーからも「最初の10秒は面白い」というフィードバックを得た。
- **テーマ統一**: 陰陽・白黒の二項対立がビジュアル（TitleBackgroundシェーダーのdomain-warped yin-yang）、メカニクス（極性切り替え）、ボスデザイン（白黒Guardian + 勾玉合体）まで一貫。
- **BGMレイヤー解放**: ゲーム進行に連動してBGMレイヤーが順次解放（Lv3でアルペジオ、90sでパッド、120sでメロディ）。インタラクティブミュージック寄りのアプローチ。
- **敵バリエーション**: 6種（Shooter/NWay/Ring/Anchor/Rush/Zoning）の敵を段階的に導入する波構成。

### 機能しなかったもの

- **3分間の持続力**: 「3分もプレイしていたいか？」というフィードバックを受けた。初見のインパクトが体験として持続しない。
- **判断空間の狭さ**: プレイヤーの能動的判断は「どこに動くか」と「いつ極性を切り替えるか」の2つだけ。パターンを理解した後は同じ判断の反復になり、30秒程度で探索し尽くされる。
- **射撃のオートエイム**: ダウンビートごとに最寄り敵へ自動発射されるため、「狙って当てた」という手応えが発生しない。敵を倒すのがプレイヤーの技術ではなく立ち位置の結果になっている。
- **成長の量的変化のみ**: レベルアップ報酬が「弾数+1」「速度+10%」で、プレイ感が質的に変わらない。弾が2発から8発に増えてもやることは同じ。
- **コンボ報酬の受動性**: 同色弾の吸収（安全な行動）でコンボが伸びるため、リスクを取る動機が薄い。「危険を冒して高得点」ではなく「安全にしていたらスコアが増えた」という体験になる。
- **短期目標の不在**: ウェーブクリア表示や進捗バーがなく、「あと少し頑張ろう」という動機が生まれにくい。

### Strategy_001との比較

前作Strategy_001（制作50時間、ゲーム開発90時間目）はテーマ「もうひとつ」をメカニクスに直結させ（すくう→溢れるリスク→もう1回？）、ループを3層（すくい判断 / ウェーブ追い / クロスラン成長）で意図的に設計できていた。

Action_002は極性切り替えというメカニクスの初見インパクトは作れたが、それを「繰り返し遊びたい体験」に展開するところまで10時間では届かなかった。

### ジャンル選択について

STG（バレットヘル）は持続的な面白さを作るのが特に難しいジャンル。弾幕の気持ちよさはパターン設計と難易度曲線の微調整に大きく依存するため、10時間ではチューニングが間に合わなかった側面がある。Strategy_001のコインすくいオートバトラーの方が、限られた時間で体験を成立させやすいジャンル選択だった。

### unity1weekの文脈

- 評価は6項目（楽しさ・絵作り・サウンド・操作性・雰囲気・斬新さ）の5段階相互評価
- 評価者は300-450作品を1週間で回すため、1作品あたり1-3分のプレイが実態
- 初見フックが効いている点はこの評価形式と相性が良い
- 一方で「楽しさ」評価は持続力の弱さが影響する可能性がある

### 次回への教訓

- 初見のインパクトは作れる。課題はそこから先の持続力。
- 判断空間を広げる設計（射撃に手動エイムを入れる、成長で質的変化を入れる等）を初期設計段階で組み込む必要がある。
- Logic Pro初触り + BGM/SFX制作に10時間を使ったのはゲームジャムとしては攻めた時間配分。全工程自作は長期的な強みだが、ジャムでは不利に働く。
- STGのような調整依存ジャンルを短時間ジャムで選ぶ場合、体験の持続力をどう担保するか事前に設計しておく必要がある。
